# 题库导入使用指南

## 功能概述

本系统提供了完整的题库导入功能，支持从 `QuestionBank.txt` 文件中导入结构化的题目和场景数据。

## 导入方式

### 方式一：UI界面导入

1. 登录系统并进入"题库管理"页面
2. 点击"完整题库导入"按钮
3. 选择 `QuestionBank.txt` 文件
4. 预览解析结果
5. 确认导入

### 方式二：控制台快速导入（推荐用于测试）

1. 打开浏览器开发者工具（F12）
2. 在控制台中运行以下命令：
   ```javascript
   quickImport()
   ```
3. 系统会自动读取 `/src/assets/QuestionBank.txt` 文件并导入

## 题库文件格式

题库文件支持以下结构：

### 渠道分类
```
## CVS
## 电商
## 大卖场/超市
## E&D
## 特殊渠道
```

### 场景题格式
```
### 场景1：场景标题
场景描述内容...

第1题（难）- 1.1从数据到洞察
题目内容...
A. 选项A
B. 选项B
C. 选项C
D. 选项D
正确答案：A
解析：
A：正确 - 解析内容
B：错误 - 解析内容
```

### 非场景题格式
```
第25题（中）- 2.3客户生意规划
题目内容...
A. 选项A
B. 选项B
C. 选项C
D. 选项D
正确答案：B
解析：
解析内容...
```

## 能力分类映射

系统支持以下7大能力分类：

1. **全域洞察力**
   - 1.1 从数据到洞察
   - 1.2 趋势预判
   - 1.3 新机会洞察

2. **客户经营力**
   - 2.1 以客户为中心
   - 2.2 潜在客户开发
   - 2.3 客户生意规划
   - 2.4 财务敏锐度

3. **故事沟通力**
   - 3.1 高效沟通技巧
   - 3.2 价值传递

4. **敏捷协作力**
   - 4.1 跨部门协作
   - 4.2 资源整合

5. **规划执行力**
   - 5.1 规划执行

6. **完美执行力**
   - 6.1 客户体验管理
   - 6.2 卓越完美执行

7. **自我驱动力**
   - 7.1 战略思维
   - 7.2 学习创新
   - 7.3 主人翁精神
   - 7.4 使用数字化工具

## 导入结果

导入完成后，系统会显示：
- 总题目数量
- 总场景数量
- 按渠道分布统计
- 按难度分布统计
- 按能力分布统计
- 验证错误报告（如有）

## 注意事项

1. 题库文件必须使用UTF-8编码
2. 题目编号必须连续且唯一
3. 每道题目必须包含完整的选项和正确答案
4. 能力分类必须符合预定义的格式
5. 导入前建议备份现有数据

## 故障排除

如果导入失败，请检查：
1. 文件格式是否正确
2. 题目编号是否连续
3. 选项格式是否规范
4. 正确答案是否有效
5. 浏览器控制台是否有错误信息
